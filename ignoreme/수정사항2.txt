피피티 수정사항
- 변수 명명 규칙, 변수 이름 정하는게 제일 힘들다
- 사람말처럼 읽히는 for문
- turtle 추가
- for문 배우고 메모장으로 코드 저장해서 도스로 실행하기
- 도스로 실행하는 법 알려주고 그 다음 도스 사용법 알려주기
- for 합불 문제 인덱스 쓰게 해서 별로 좋은 문제는 아님.
- with 그냥 점프 투 파이썬처럼 밖으로 빼자.

https://docs.python.org/ko/3/reference/compound_stmts.html#the-for-statement
for i in a:
    if x < 0: a.remove(x)
하면 a에서 x가 지워졌을 때, 인덱스가 앞으로 땡겨지기 때문에 다음 항목을 건너뛰게 됨. 여기서 말하는 "내부 카운터"가 인덱스 단위여서 그런듯. 단순 카피로 해결.
for i in a[:]:
    if x < 0: a.remove(x)

https://docs.python.org/ko/3/reference/compound_stmts.html#compound-statements
if x < y < z: print(x); print(y); print(z)
;이 :보다 강하게 결합해서
if x < y < z:
    print(x)
    print(y)
    print(z)
와 같음. 표현식을 저렇게 모호하게 쓰지는 말자.
